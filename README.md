# 简介
将dubbo服务对外暴露`http`服务。

可供其他任何语言进行调用。


# 安装

```
git clone https://github.com/crossoverJie/SSM-DUBBO-HTTP.git
```

```
cd SSM-DUBBO-HTTP
```

```
mvn clean
```

```
mvn install
```


# 使用

```xml
<dependency>
    <groupId>com.crossoverJie</groupId>
    <artifactId>SSM-HTTP-PROVIDER</artifactId>
    <version>1.0.0</version>
</dependency>
```
